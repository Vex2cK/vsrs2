## moscow_metro.csv и other_cities

- station_name: название станции

- station_lat: широта

- station_lng: долгота

- line_id: номер линии

- line_name: название линии

- station_id: уникальный идентификатор станции (вместо названия)

- station_order: порядковый номер станции на линии
